## ✨ 핵심 기능

1. **🟢 인터랙티브 트렌드 버블 차트 (SVG + Pure JS Physics)**
   - 비율 지수가 높은 키워드는 **큰 원**, 낮은 키워드는 **작은 원**으로 자동 크기 가변화되어 표시됩니다.
   - SVG 물리 충돌 및 부유 모션이 탑재되어 원들이 서로 겹치지 않고 자연스럽게 부유합니다.
   - 상단 카테고리 필터링 탭을 전환하면 매칭되는 버블들만 확대/축소 모션을 거치며 실시간으로 모여듭니다.

2. **📊 세부 분석 패널**
   - 버블을 클릭하면 우측 하단 상세 정보창에 실시간 트렌드 추이 그래프(SVG Sparkline) 및 연관 워드클라우드 태그가 업데이트됩니다.
   - 검색창을 통해 원하는 키워드를 입력해 추가하면, 해당 키워드가 즉시 원형 버블이 되어 통통 튀며 화면에 시뮬레이션 렌더링됩니다.
